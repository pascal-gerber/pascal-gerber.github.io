Important CSS lesson

In css there are 3 formating supports

Margin = Red
Border = Green
Padding = yellow

The maring contains directly the text, the border can be colored or so to make a good looking border and the margin is just there to place correctly the widget.

    you can use any of those 12 commands

  • Margins:

    1. margin-top: 20px;
    2. margin-left: 20px;
    3. margin-bottom: 20px;
    4. margin-right: 20px;

    margin: 20px 20px 20px 20px;

    margin: 20px; (since thier all 4 the same)

  • Borders:

    1. border-top: 20px
    2. border-left: 20px
    3. border-bottom: 20px;
    4. border-right: 20px;

    border: 20px 20px 20px 20px;

    border: 20px; (since thier all 4 the same)

  • paddings:

    1. padding-top: 20px;
    2. padding-left: 20px;
    3. padding-bottom: 20px;
    4. padding-right: 20px;

    • padding: 20px 20px 20px 20px;

    padding: 20px; (since thier all 4 the same)
